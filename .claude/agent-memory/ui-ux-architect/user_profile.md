## Role
Owner/developer of ProTrader — a personal trading intelligence dashboard. Likely uses it daily for investment decisions.

## Design Goals
- Apple-style: Clean, minimal, lots of whitespace, refined typography, subtle depth via shadows
- Sleek: Smooth transitions, refined surfaces
- Trustworthy: Professional palette, clear hierarchy, confidence-inspiring
- Inviting: Warm tones, approachable, not intimidating
- Different colors: Move away from GitHub-dark cold palette

## Context
- Single-user app (one trader)
- Used for: mutual fund analysis, stock picks, automated trading overview, crypto tracking
- Data-dense by nature (trading data) — must balance density with readability
- Built on React + Vite + Tailwind + CSS custom properties
